# var
> 无类型的概念。



### 多个变量赋值
Python允许你同时为多个变量赋值。例如：

a = b = c = 1
以上实例，创建一个整型对象，值为1，三个变量被分配到相同的内存空间上。

### 解析赋值
您也可以为多个对象指定多个变量。例如：

a, b, c = 1, 2, "john"
以上实例，两个整型对象 1 和 2 分别分配给变量 a 和 b，字符串对象 "john" 分配给变量 c。



### 标准数据类型
- Numbers（数字）
- String（字符串）
- List（列表）
- Tuple（元组）
- Dictionary（字典）
